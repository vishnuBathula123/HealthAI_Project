# HealthAI Project

HealthAI is an AI-based healthcare assistant developed using Python and Streamlit.
The application helps users analyze symptoms, identify possible medical conditions,
and receive basic treatment and lifestyle guidance.

## Features
- Symptom-based disease prediction
- Emergency symptom detection
- Chat-based healthcare assistant
- Treatment and lifestyle recommendations
- Health analytics visualization (sample data)

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy

## How to Run the Project
1. Install required libraries:
   pip install streamlit pandas numpy

2. Run the application:
   streamlit run app.py

professional medical advice.
